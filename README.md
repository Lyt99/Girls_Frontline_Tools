## 目录及文件说明


### Data
用于存放游戏数据的目录

#### gun\_data.xlsx
枪的数据

#### main\_quest\_data.xlsx
主线任务数据

#### daily\_quest\_data.xlsx
日常任务数据

#### weekly\_quest\_data.xlsx
周常任务数据

#### develop\_timetable.xlsx
从gun\_data.xlsx里提取出来的建造时间表，方便取用

#### mission\_info.xlsx
战役相关数据

#### achievement\_info.xlsx
成就数据

#### equip\_timetable.xlsx
装备建造时间表


#### TextAsset\
剧情和教程指导文件
